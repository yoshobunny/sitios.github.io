01 — Concepto central

¿Qué es
Machine Learning?

🧑‍💻 Programación Tradicional

  • Tú escribes las reglas
  • Datos + Reglas → Resultado
  • Si cambian los datos, reescribes
  • Frágil ante lo inesperado
VS

🤖 Machine Learning

  • El modelo aprende las reglas
  • Datos + Resultado → Reglas
  • Se adapta a nuevos datos
  • Mejora con la experiencia

💡 Analogía: No le dices a un niño "si tiene 4 patas, rabo y ladra = perro". Le muestras 1,000 fotos de perros y él construye sus propias reglas.

02 — Taxonomía

3 Tipos de
Aprendizaje

🏷️

Supervisado

Aprende con ejemplos etiquetados. Tiene respuestas correctas durante el entrenamiento.

→ Clasificación, Regresión
🔍

No Supervisado

Encuentra patrones ocultos en datos sin etiquetas. El modelo descubre la estructura.

→ Clustering, Reducción de dimensión
🎮

Por Refuerzo

Aprende por ensayo y error. Maximiza una recompensa interactuando con un entorno.

→ Juegos, robótica, recomendaciones

🎯 En este Certificado nos enfocamos en Supervisado y No Supervisado.

🏷️ Features vs Labels

Feature = lo que sabes (edad, historial, comportamiento)

Label = lo que quieres predecir (¿compra? ¿fraude? ¿churn?)

⚠️ La trampa del Overfitting

Como el estudiante que memoriza el examen pero no entiende el tema. Perfecto en training, pésimo en datos nuevos.
